## MODIFIED Requirements

### Requirement: Modern Sidebar Navigation
The system SHALL replace the existing semi-transparent sidebar with a modern, solid-colored sidebar based on the provided design.

#### Scenario: Visual and Interactive improvements
- **WHEN** navigating the application
- **THEN** the sidebar SHALL show the active route with a highlighted background.
- **AND** the sidebar SHALL use the `LayoutDashboard`, `FileSpreadsheet`, `BookOpen`, and `Wand2` icons from Lucide.
- **AND** the sidebar SHALL include a header title "Excel Data Cleaner".
