## MODIFIED Requirements

### Requirement: Dictionary Overview and Management
The Dictionary page SHALL provide an overview of dictionary statistics and storage management, and allow bulk actions such as multi-activation or simultaneous processing.

#### Scenario: View Dictionary Stats
- **WHEN** a user navigates to the Dictionary page
- **THEN** the system SHALL display usage statistics and a list of available dictionaries for management.

#### Scenario: Multi-dictionary Selection
- **WHEN** a user selects multiple available dictionaries from the management list
- **THEN** the system SHALL allow the user to activate or deactivate the selected dictionaries as a bulk operation.
